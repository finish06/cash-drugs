drug-cash

Drug data API cache — fast, reliable, always available

Fetches from DailyMed, FDA openFDA, and RxNorm APIs. Caches responses in MongoDB. Serves internal consumers in < 50ms. When upstreams go down, stale data keeps you running.

API Endpoints

GET /api/cache/{slug}

Retrieve cached data by endpoint slug. Returns paginated results with freshness metadata.

{
  "data": [
    {
      "product_ndc": "0002-1433",
      "brand_name": "Metformin Hydrochloride",
      "dosage_form": "TABLET",
      "route": "ORAL"
    }
  ],
  "meta": {
    "slug": "fda-ndc",
    "source_url": "https://api.fda.gov/drug/ndc.json",
    "fetched_at": "2026-03-24T08:15:00Z",
    "page_count": 12,
    "results_count": 1187,
    "stale": false
  }
}

GET /api/search?q=metformin

Cross-slug full-text search. Returns results grouped by source with match counts.

{
  "query": "metformin",
  "total_matches": 42,
  "results": [
    {
      "slug": "fda-ndc",
      "matches": 28,
      "items": [{"brand_name": "Metformin HCl", "product_ndc": "0002-1433"}]
    },
    {
      "slug": "drugnames",
      "matches": 14,
      "items": [{"name": "Metformin Hydrochloride 500mg Tablet"}]
    }
  ],
  "duration_ms": 23.4
}

GET /api/autocomplete?q=met&limit=5

Fast prefix-match autocomplete for typeahead UIs. Returns matching drug names.

{
  "query": "met",
  "suggestions": [
    "Metformin Hydrochloride",
    "Metoprolol Succinate",
    "Methotrexate",
    "Methylphenidate",
    "Metronidazole"
  ]
}

GET /api/endpoints

Discover all configured upstream endpoints with slug names, URLs, and cache status.

{
  "endpoints": [
    {"slug": "fda-ndc", "url": "https://api.fda.gov/drug/ndc.json", "cached": true},
    {"slug": "drugnames", "url": "https://dailymed.nlm.nih.gov/...", "cached": true},
    {"slug": "fda-enforcement", "url": "https://api.fda.gov/drug/enforcement.json", "cached": true}
  ]
}

GET /api/cache/{slug}/_meta

Per-slug metadata: freshness, document count, last fetch time, staleness info.

{
  "slug": "fda-ndc",
  "document_count": 1187,
  "last_fetched": "2026-03-24T08:15:00Z",
  "ttl_seconds": 86400,
  "stale": false,
  "cache_age_seconds": 3420
}

POST /api/cache/{slug}/bulk

Bulk cache lookups. Send multiple parameter sets, get results in one request.

// Request
{"queries": [{"search": "metformin"}, {"search": "lisinopril"}]}

// Response
{
  "results": [
    {"params": {"search": "metformin"}, "count": 28, "data": [...]},
    {"params": {"search": "lisinopril"}, "count": 15, "data": [...]}
  ],
  "total_queries": 2,
  "duration_ms": 45.2
}

Quick Start

# Clone and run
git clone https://github.com/finish06/cash-drugs.git
cd cash-drugs
docker compose up

# API available at http://localhost:8080
# Fetch cached drug data
curl http://localhost:8080/api/endpoints
curl http://localhost:8080/api/cache/fda-ndc
curl "http://localhost:8080/api/search?q=metformin"
